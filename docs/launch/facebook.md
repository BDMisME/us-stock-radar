📡 我做了一個美股 AI 看盤雷達，現在開源了，免費分享給大家 👇

身為一個有正職的人，盤中根本沒辦法一直盯美股。但又不想錯過手上持股的重要變化——跌破均線、爆量、接近停損停利、進入我設定的買區……這些時刻才是真正要看的。

於是我做了 **us-stock-radar**：一個跑在自己機器（或雲端）上的個人看盤儀表板。它幫我做三件事：

1️⃣ **自動盯盤**：接 Alpaca 即時行情，自算 MA / RSI / MACD / 量比等指標，用「規則引擎」偵測 15 種異常事件。
2️⃣ **AI 幫我消化**：重要事件才丟給 AI 分析師（火山方舟）做**結構化研究與風險提醒**——它會講「續抱 / 減碼 / 觀察買點」、附上**失效條件**，而且有冷卻機制不會洗版。
3️⃣ **重大事件通知我**：Telegram / Email 主動提醒，不用一直開著看。

幾個我自己很喜歡的細節：
🔴🟢 **台股慣例配色**（紅漲綠跌），看起來最直覺
📊 持股**熱力圖**，一眼看出今天誰在動、部位集中在哪
📈 K 線可**拖曳平移、滾輪縮放**，還有時間區間快捷鈕
🎯 觀察名單會標**「在買區 / 低於 / 高於」**目標價

⚠️ 必須講清楚：**這個工具不會幫你下單、不串任何券商交易 API、不保證獲利**。AI 只做研究與風險提醒，**不是投資建議**，所有決策請自己判斷、自己負責。我做它是為了「更好地看」，不是為了「自動交易」。

技術上是 Python + Streamlit + SQLite，架構很輕量（不用 Redis/Kafka），所有 API key 都是選填、缺了會優雅降級。想自架的人 clone 下來、`pip install`、填好自己的 key 就能跑；只想看畫面的話全部留空也有 demo 資料。
